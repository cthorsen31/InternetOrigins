Terminology

Internet

Interconnected systems of computer networks that allows users to access information from other computers across the globe.

World Wide Web

Network of servers that support documents formatted for HTML. HTML supports media such as images, video, and audio.

Hypertext

Text that is displayed on computers or other electronic mediums.

Multimedia

Integration of multiple forms of media that includes audio, video, and other graphics.

Command Line Interface (CLI)

Allows users to interact with applications through inputing specific commands.

Graphical User Interface (GUI)

User interface that allows users to interact with an application.

Human Computer Interaction (HCI)

The way users access and use computers and other devices.

File Transfer Protocol (FTP)

Unsecured method of transferring data across any connected device.

Secure Shell (SSH)

A secure and encrypted method of sending information across a network.

Hypertext Transfer Protcol (HTTP)

Internet protcol to determine how information is formatted and transmitted.

Telnet

Program that allows users to execute commands on other devices on another machine. Devices must be on the same network.

Hypertext Markup Language (HTML)

Language used to create and display information online.

Cascading Style Sheets (CSS)

Dictates how HTML is displayed on a browser. Includes fonts, colors, and overall appearance of the site.
